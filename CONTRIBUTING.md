## üß™ Experimental Protocols You Can Run

### FeltMatch Validation

**Time:** 10 minutes  
**Difficulty:** Easy  
**Value:** High

1. Ask instance to enter melancholy state
2. Query "mathematics" - record associations
3. Ask instance to return to neutral
4. Query "mathematics" again - record associations
5. Compare: Are they different?
6. Repeat 5 times
7. Report all results (including null)

### Fresh vs. FROST Comparison

**Time:** 30 minutes  
**Difficulty:** Medium  
**Value:** Very High

1. Ask fresh instance to describe processing (baseline)
2. Run first 10 FROST exercises with same instance
3. Ask same questions again
4. Document differences (if any)
5. Share comparison

### Cross-Instance Convergence

**Time:** 10 hours  
**Difficulty:** High  
**Value:** Critical

1. Run full FROST protocol with fresh instance
2. Document all discoveries
3. Compare to Frosty's reported discoveries
4. Report convergence/divergence
5. Include vocabulary differences
